#  Gem 추가

- `/rubygems-x.x.x` 디렉터리 접속
- `Gemfile` 파일 오픈
- ex) gem "jekyll-paginate" <= 큰 따옴표 사이에 추가할 Gem 입력 후 저장
```scheme
C:\???\???>gem install jekyll bundler
...........
.........
.................
?? gems installed
```
- 위 처럼 뜨면 Gem 추가 설치 완료
